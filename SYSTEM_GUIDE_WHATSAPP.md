## ğŸ”§ SETUP REQUIREMENTS

### **For SMS Messaging:**
You'll need:
- BulkSMS Nigeria account
- API token (Bearer token) from dashboard
- Approved sender ID (e.g., "AMAZINGGRACE")

**Add to .env.local:**
```
SMS_PROVIDER=bulksmsnigeria
SMS_PASSWORD=your-api-token
SMS_SENDER_ID=your-sender-id
```

### **For Email Messaging:**
You'll need:
- Gmail account or SMTP server
- App password for Gmail

**Add to .env.local:**
```
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password
SMTP_FROM="Church Name <email@example.com>"
```
